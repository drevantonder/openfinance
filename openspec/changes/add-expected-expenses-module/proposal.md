# Change: Add expected expenses module

## Why
People need a clear, simple way to set expected expenses and one-off savings targets so they can plan ahead without duplicating income setup.

## What Changes
- Introduce an expected expenses module for expense targets and schedules
- Add recurring and once-off expense target schedules
- Add savings targets managed as expense targets
- Expose read-only target hooks for other modules to consume

## Impact
- Affected specs: expected-expenses
- Affected code: expected expenses module UI, API, storage
