# Testing

This doc summarizes the current testing bar and how to run it.

## Source of truth

Current truth for testing lives in `docs/specs/project-setup/spec.md`.

## Test bar

- Run `pnpm lint`, `pnpm test`, and `pnpm build` before merging changes.
- Add coverage only when a change introduces new behavior or fixes a bug.

## Commands

- `pnpm lint`
- `pnpm test`
- `pnpm build`
